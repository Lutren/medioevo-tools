# DECISIONS v1.0

- Use Canvas 2D and procedural geometry; no WebGPU or path tracing.
- Preserve the existing renderer and layer pixel realism passes around it.
- Use internal logical resolutions: 80x45, 160x90 and 320x180.
- Treat light physics as physically inspired approximation only.
- Keep VibeCoding deterministic and local.
- Keep Wabi disabled and design-only.
