# GENERATIVE_PIXEL_ART_ENGINE_v1_2

## Implemented

Procedural generators produce deterministic pixel-art fallback bitmaps for tiles, buildings, props, UI, agents and composed scenes.

Modules:

- `src/generative/pixelArtGenerator.ts`
- `src/generative/proceduralTileGenerator.ts`
- `src/generative/proceduralBuildingGenerator.ts`
- `src/generative/proceduralPropGenerator.ts`
- `src/generative/proceduralUiGenerator.ts`
- `src/generative/proceduralAgentGenerator.ts`
- `src/generative/generativeSceneComposer.ts`

## Assets

No external AI, cloud, API or copied assets are used. Style is driven by local tokens and deterministic seeds.
