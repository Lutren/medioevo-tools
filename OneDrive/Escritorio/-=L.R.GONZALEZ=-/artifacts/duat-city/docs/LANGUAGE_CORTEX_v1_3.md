# LANGUAGE_CORTEX_v1_3

The Language Cortex is deterministic local code. It does not call external AI, API or cloud services.

## Capabilities

- NPC speech from role, task, memory, mood and R/Phi.
- Epistemic classification: CERTEZA, INFERENCIA, INCOGNITA, BLOQUEO.
- Unsupported claims are downgraded by `languageGate`.
- Source cards compile from city events.
- Vibe commands map to local game actions.
- Metrics: R_language, Phi_language, ambiguity, unsupported_claims, repetition, narrative_coherence.

## Boundary

NPCs speak from local causal knowledge only. They are not omniscient.
