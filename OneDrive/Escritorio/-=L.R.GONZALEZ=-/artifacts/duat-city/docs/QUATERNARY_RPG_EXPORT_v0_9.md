# QUATERNARY RPG EXPORT v0.9

## Export Shape

`medioevo-rpg/world/v2` now includes:

```json
{
  "quaternary_profile": {
    "counts": {},
    "anomaly_zones": [],
    "stable_zones": [],
    "event_zones": [],
    "unstable_agents": [],
    "suggested_quests": []
  }
}
```

## Quest Hooks

The profile can create hooks for:

- `01` around ruins/gatehouse: investigate missing signal;
- high-frequency market signals: stabilize market oscillation;
- long `10` archive channels: recover stable knowledge;
- repeated `11` observatory/event zones: follow active anomaly.

The export remains local and schema-compatible with the existing v2 world export.
