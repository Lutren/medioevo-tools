# RPG_EXPORT_V3_v1_2

## Schema

RPG export now emits:

`schema: medioevo-rpg/world/v3`

## Added Fields

- `art_direction_profile`
- `style_tokens`
- `asset_references`
- `procedural_generation_seeds`
- `material_field_summary`
- `light_field_summary`
- `vibe_command_history`
- `agent_routes`
- `interaction_points`

## Boundary

The art profile contains MEDIOEVO-original tokens and `public_boundary_note`. Asset references point to manifests with `publication_allowed=false`; no copied v1.2 asset is treated as public-safe.
