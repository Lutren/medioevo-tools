## Core Rule

Anything released under MIT must be treated as actually MIT:

- anyone can use, copy, modify, merge, publish, distribute, sublicense and sell
  copies under the MIT terms;
- MEDIOEVO cannot publish code as MIT and then require top-tier membership to
  exercise MIT rights;
- tiers can sell curation, support, packaging, templates, early previews,
  briefings and unreleased/private material.

## Public MIT Candidate Criteria

A package can be described as MIT/open only when all are true:

- it has its own `LICENSE`;
- it has target-specific README and claims boundaries;
- it excludes private game, books, raw research and local runtime;
- it uses synthetic fixtures or approved public-safe examples;
- secret scan on the exact target reports no blocking findings;
- path scrub finds no private local paths;
- claims scan finds no strong unverified claims;
- ActionGate approves the exact publication target.

## Tier Copy Rule

Correct:

```text
The code foundations are MIT and free. Sponsors receive curated research, reviewed demos, briefings, support and early public-safe material.
```

Incorrect:

```text
This MIT code is open source but only top-tier sponsors can use it commercially.
```

Correct:

```text
Top-tier members receive reviewed Observacionismo deconstructions and transfer maps. Raw formulas, private runtime and unreleased canon stay private.
```

Incorrect:

```text
Top-tier members get the full private system and all raw research.
```
