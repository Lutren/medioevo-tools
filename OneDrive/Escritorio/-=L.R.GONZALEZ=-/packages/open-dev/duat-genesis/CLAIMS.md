# Claims

## Allowed

- DUAT Genesis is a synthetic, deterministic simulation sandbox.
- It can help prototype observation, rule pressure, reporting and falsifier
  patterns.
- Its examples are reproducible for the same seed and inputs.

## Prohibited

- It proves Observacionismo, PSI, EML, MCR or any physics theory.
- It predicts society, neurology, bacteria, proteins, medicine or markets.
- It contains DUAT Geodia private engineering.
- It contains private MEDIOEVO RPG/TCG technology, assets or lore.
- It can be used as a medical, legal, financial or safety system without domain
  validation.

## Current Evidence

- Unit tests cover deterministic simulation, bounded state, report shape,
  falsifier output and CLI JSON output.
- Unit tests cover Run 8 public helpers: module registry validation, ActionGate
  v2 decisions, handoff validation, public prompts, source cards, WitnessLog
  JSONL shape and legacy checklist shape.
- Calibration is `DEMO_ONLY`.
